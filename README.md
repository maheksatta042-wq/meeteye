
  # Employee Tracking Dashboard (Frontend)

  This is a code bundle for Employee Tracking Dashboard. The original project is available at https://www.figma.com/design/VDHxAbT6GXOffpfzFRX7Pq/Employee-Tracking-Dashboard.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.


  For deployement in Render always remove node_module folder from repo and let the render install all dependencies as it will block the permission to install all dependencies in render.
